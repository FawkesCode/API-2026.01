# User Story 11

**Como** diretor, **quero** visualizar um dashboard gerencial com indicadores de pedidos, gastos por centro de custo e alertas de estoque, **para** tomar decisões estratégicas com base em informações atualizadas.

---

## Critérios de Aceite (CA)

### CA1: Visualização de indicadores de pedidos de compra
**Dado** que sou um diretor autenticado no sistema  
**Quando** eu acesso o dashboard gerencial  
**Então** o sistema exibe indicadores de pedidos de compra:  
**E** total de pedidos por status (pendente, aprovado, reprovado, em cotação, concluído)  
**E** valor total por status  
**E** quantidade de pedidos pendentes de minha aprovação  
**E** tempo médio de aprovação de pedidos  
**E** taxa de aprovação vs reprovação (percentual)  
**E** os dados são atualizados em tempo real ou periodicamente

### CA2: Visualização de gastos por centro de custo
**Dado** que sou um diretor autenticado no sistema  
**Quando** eu visualizo a seção de gastos no dashboard  
**Então** o sistema exibe gastos agrupados por centro de custo:  
**E** valor total gasto por centro de custo no período selecionado  
**E** percentual de cada centro de custo em relação ao total  
**E** gráfico visual (pizza ou barras) para comparação  
**E** top 5 centros de custo com maiores gastos  
**E** posso filtrar por período (mês atual, último trimestre, ano, personalizado)

### CA3: Visualização de alertas de estoque
**Dado** que sou um diretor autenticado no sistema  
**Quando** eu visualizo a seção de estoque no dashboard  
**Então** o sistema exibe alertas de estoque:  
**E** quantidade de itens abaixo do nível mínimo  
**E** lista dos itens críticos (com quantidade atual e nível mínimo)  
**E** valor estimado para reposição dos itens críticos  
**E** alertas destacados visualmente (cores, ícones)  
**E** posso clicar em um alerta para ver detalhes do item

### CA4: Filtros de período personalizáveis
**Dado** que sou um diretor visualizando o dashboard  
**Quando** eu seleciono um filtro de período (hoje, última semana, último mês, último trimestre, ano, personalizado)  
**E** aplico o filtro  
**Então** todos os indicadores são recalculados e atualizados para o período selecionado  
**E** gráficos e números refletem os dados do período filtrado  
**E** o filtro aplicado é claramente visível no dashboard

### CA5: Gráficos interativos e visuais
**Dado** que sou um diretor visualizando o dashboard  
**Quando** eu observo os indicadores  
**Então** o sistema exibe gráficos visuais e interativos:  
**E** gráfico de barras ou linhas para evolução de pedidos ao longo do tempo  
**E** gráfico de pizza para distribuição de gastos por centro de custo  
**E** gráfico de status de pedidos  
**E** indicadores com ícones e cores para fácil interpretação  
**E** posso clicar em elementos do gráfico para drill-down (detalhamento)

### CA6: Exportação de relatório gerencial
**Dado** que sou um diretor visualizando o dashboard  
**Quando** eu clico na opção "Exportar Relatório"  
**E** seleciono o formato (PDF ou Excel)  
**Então** o sistema gera um relatório com todos os indicadores visíveis no dashboard  
**E** o relatório inclui gráficos e tabelas  
**E** o relatório é baixado automaticamente  
**E** o relatório contém data de geração e período filtrado

### CA7: Atualização automática de dados
**Dado** que sou um diretor visualizando o dashboard  
**E** novas operações são realizadas no sistema (aprovações, criação de pedidos, movimentações de estoque)  
**Quando** os dados são alterados  
**Então** o dashboard é atualizado automaticamente (ou com refresh manual)  
**E** um indicador visual mostra que os dados foram atualizados  
**E** não é necessário recarregar a página completa
