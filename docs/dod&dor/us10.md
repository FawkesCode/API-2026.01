# User Story 10

**Como** gerente, **quero** realizar cotações comparando propostas de múltiplos fornecedores para um mesmo pedido, **para** selecionar a opção mais vantajosa antes de efetuar a compra.

---

## Critérios de Aceite (CA)

### CA1: Criação de cotação vinculada a pedido aprovado
**Dado** que sou um gerente autenticado no sistema  
**E** existe um pedido de compra aprovado  
**Quando** eu acesso a tela de criação de cotação  
**E** seleciono o pedido de compra a ser cotado  
**E** clico em "Criar Cotação"  
**Então** o sistema cria uma nova cotação vinculada ao pedido  
**E** gera um número único para a cotação  
**E** o status do pedido muda para "Em Cotação"  
**E** uma mensagem de sucesso é exibida

### CA2: Adição de propostas de múltiplos fornecedores
**Dado** que sou um gerente com uma cotação criada  
**Quando** eu acesso a tela de gerenciamento da cotação  
**E** adiciono uma proposta informando fornecedor, prazo de entrega, condições de pagamento e valores por item  
**E** clico em "Adicionar Proposta"  
**Então** a proposta é vinculada à cotação  
**E** posso adicionar propostas de outros fornecedores para o mesmo pedido  
**E** cada proposta exibe o valor total calculado automaticamente

### CA3: Comparação visual de propostas
**Dado** que sou um gerente visualizando uma cotação com múltiplas propostas  
**Quando** eu acesso a tela de comparação  
**Então** o sistema exibe uma tabela comparativa com todas as propostas lado a lado  
**E** mostra para cada fornecedor: valor total, prazo de entrega, condições de pagamento, valores individuais por item  
**E** destaca visualmente a proposta com menor valor total  
**E** destaca visualmente a proposta com menor prazo de entrega  
**E** permite ordenar propostas por critérios (valor, prazo, fornecedor)

### CA4: Seleção da proposta vencedora
**Dado** que sou um gerente visualizando propostas de uma cotação  
**Quando** eu seleciono uma proposta como vencedora  
**E** registro uma justificativa da escolha  
**E** clico em "Confirmar Seleção"  
**Então** a proposta é marcada como vencedora  
**E** a justificativa é registrada  
**E** o status da cotação muda para "Concluída"  
**E** o status do pedido muda para "Cotado"  
**E** as demais propostas são marcadas como "Não Selecionadas"  
**E** uma mensagem de sucesso é exibida

### CA5: Validação de campos obrigatórios ao adicionar proposta
**Dado** que sou um gerente adicionando uma proposta  
**Quando** eu tento adicionar sem preencher campos obrigatórios (fornecedor, prazo de entrega, valores dos itens)  
**Então** o sistema exibe mensagens de erro indicando os campos não preenchidos  
**E** a proposta não é adicionada até que todos os campos sejam preenchidos

### CA6: Validação de valores não negativos nas propostas
**Dado** que sou um gerente adicionando valores a uma proposta  
**Quando** eu tento informar valores negativos ou zero para itens  
**Então** o sistema exibe uma mensagem de erro "Os valores devem ser maiores que zero"  
**E** não permite salvar a proposta

### CA7: Cancelamento de cotação
**Dado** que sou um gerente com uma cotação em andamento  
**Quando** eu seleciono a opção "Cancelar Cotação"  
**E** registro um motivo do cancelamento  
**E** confirmo a ação  
**Então** o status da cotação muda para "Cancelada"  
**E** o status do pedido volta para "Aprovado"  
**E** o motivo do cancelamento é registrado  
**E** as propostas são mantidas para histórico mas marcadas como inativas
